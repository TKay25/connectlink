3 Sending a Text Message

✉️ Sending
Hi John, confirming tomorrow's site visit at 9am.
Sending... ⏳
😀📎 Type a message... Send
  1. Select a chat from the sidebar.
  2. Type your message in the input bar at the bottom.
  3. Press Enter or click the Send button.
  4. A pending bubble with ⏳ appears, then updates to ✓ when sent.
24hr window: Free-form messages only work within 24 hours of the client's last message. Otherwise use a Meta Template (Section 6).

  • 📷 Images: JPG, PNG, GIF
  • 🎵 Audio: MP3, M4A, OGG

  1. Click the 📎 paperclip icon in the input bar.

  5. Audio files are sent as downloadable documents for reliable playback.

5 Bot Mode vs Human Mode

🤖 Bot / Human Toggle
Bot Human
Human mode ON: AI auto-replies disabled. Full manual control.
  1. Each chat has a toggle switch in the header: Bot ⇄ Human.
  2. Bot mode (default) — the AI chatbot auto-replies to incoming messages.
  3. Human mode — the AI does NOT reply; you handle the conversation manually.
  4. Toggle anytime — setting is saved per-contact in the database.

6 Meta Templates (24hr Window Bypass)

📋 Template Mode
📋 Template Message
enquiryupdate sent to John
✓ Delivered via Meta template
Common templates:
  • enquiryupdate
  • quotationdownload
  • overduereminderfin
  • duereminderfin
  1. Click the Tpl button next to the input bar to switch to template mode.
  2. Enter the exact template name as approved in Meta (e.g., enquiryupdate).
  3. Enter parameters as comma-separated values (e.g., John,Project X,50).
  4. Click Send — templates bypass the 24-hour window.
  5. To switch back, click Text.
Tip: Template names must match Meta's catalog exactly. Wrong names will fail.

10 Bulk Messaging

📤 Bulk
1. Select recipients using checkboxes. Use Select all or the filter box.
2. Compose: Header (bold), Body (required), Footer (small), optional 📷 image.
3. Preview updates live. Click Send Bulk → confirm popup → sent.
Preview: Header in bold → Body text → Footer in gray. Failures show per-recipient error details.
  1. Click the Bulk tab. Select recipients via checkboxes.
  2. Enter Header (optional bold), Body (required), Footer (optional small text).
  3. Optionally attach an image using the 📷 button.

  5. Click Send Bulk → review the confirmation popup → Confirm Send.

15 WhatsApp Calls

📞 Call
📞 Click the phone icon in chat header → opens wa.me/{number} in new tab
  1. Open any chat — the 📞 icon appears in the header.

WhatsApp Cloud API does not support initiating calls directly. The deep link opens WhatsApp where you can call manually.

19 Troubleshooting

Messages not sending? Check if you're outside the 24-hour window. Use a Meta Template instead.
File send failed? File may be too large (>16MB) or unsupported format.
Chat list not loading? Check server is running and DB is connected. Press F12 → Console tab for errors.
Notifications not appearing? Ensure notification permission is granted. iOS PWA notifications have limited support.
Need help? Check the browser console (F12) for errors and share them with support.
